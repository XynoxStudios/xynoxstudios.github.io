FREQUENTLY ASKED QUESTIONS

How do I join InsaneSMP?

+

Copy the server IP and connect via Minecraft Java or Bedrock. Make sure your account is registered.

Is InsaneSMP free to play?

+

Yes! Server access is free. Optional cosmetics are available but no pay-to-win mechanics exist.

How do I report a bug?

+

Report bugs via Discord or our GitHub tracker. Include screenshots/video and reproduction steps.
